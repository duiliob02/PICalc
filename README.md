# PICalc
Estimating Ï€ digits to learn about any performances improvements with Swift language about a piece of code written with Concurrency versus his Sequential counterpart.

## How to use
Right after the repo has been cloned, through CLI within the folder of the package run

```
swift run PICalc 0
```

to start the program with the sequential Execution, while running

```
swift run PICalc 1
```

will start the Parallel execution of the code.
